# Criar template RFQ

Use `*criartemplateRFQ` para criar um modelo-base reutilizavel.

## O que o template deve congelar
- tipo de demanda
- tom comercial
- estrutura do assunto
- blocos fixos do corpo
- placeholders para descricao tecnica, prazo, quantidade e documentacao

## Saida
- nome do template
- quando usar
- template completo com placeholders claros
