# Follow-up RFQ

Use `*followupRFQ` para devolver um follow-up pronto para copiar.

## Estrutura
- Assunto
- Saudacao
- Referencia ao RFQ anterior
- Pedido claro de retorno
- Fechamento curto

## Regras
- Ser mais curto que o RFQ inicial.
- Cobrar resposta sem agressividade.
- Se houver prazo, explicitar.
