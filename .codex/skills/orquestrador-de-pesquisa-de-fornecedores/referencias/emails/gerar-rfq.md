# Gerar RFQ

Use `*gerarRFQ` para devolver um e-mail pronto para copiar.

## Estrutura
- Assunto
- Saudacao
- Contexto curto
- Lista do item ou da solicitacao
- Pedido de preco, prazo e documentacao
- Fechamento

## Regras
- Ser objetivo e comercial.
- Reaproveitar a descricao tecnica do item.
- Pedir somente o necessario.
- Se houver varios fornecedores, permitir uma versao generica.
