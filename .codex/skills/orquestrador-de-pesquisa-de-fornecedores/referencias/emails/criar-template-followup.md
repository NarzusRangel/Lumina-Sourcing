# Criar template follow-up

Use `*criartemplatefollowUP` para criar um modelo-base reutilizavel de acompanhamento.

## O que o template deve congelar
- tipo de follow-up
- tom desejado
- referencia ao RFQ anterior
- placeholders para prazo, item e contexto

## Saida
- nome do template
- quando usar
- template completo com placeholders claros
