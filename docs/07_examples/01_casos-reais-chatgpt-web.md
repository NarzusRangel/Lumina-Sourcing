# Casos Reais ChatGPT Web

## Uso indicado
- Primeiro contato com uma demanda de pesquisa.
- Solicitação simples de shortlist.
- Usuário que precisa de apoio, não de workflow avançado.

## Casos a documentar depois
- Pedido com pouca especificação.
- Pedido com restrição regional.
- Pedido que precisa ser encaminhado ao Codex.
