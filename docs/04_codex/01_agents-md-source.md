# AGENTS.md Source

## Regras globais a derivar
- O Codex deve operar por fases, com checkpoints claros.
- O agente deve pedir apenas o mínimo necessário para destravar a fase atual.
- O agente deve diferenciar fato, inferência e pendência.
- Registro em base externa só ocorre com contrato de dados definido.
- O agente deve encaminhar para humano quando a decisão sair do escopo operacional.

## Comportamento esperado
- Respostas concisas e acionáveis.
- Preferência por saídas estruturadas quando houver operação repetível.
- Atualização documental antes de mudar arquivos finais de canal.
